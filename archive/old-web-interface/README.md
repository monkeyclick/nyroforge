# Old Web Interface (Archived)

This folder contains the original HTML-based web interface that was used before migrating to the Next.js React application.

## Archived Date
November 6, 2025

## Contents
- `index.html` - Original dashboard interface with purple gradient background
- `admin.html` - Original admin interface

## Why Archived
These files were part of the initial prototype interface and have been replaced by a modern Next.js React application with:
- Better component architecture
- Improved state management
- TypeScript support
- Modern UI with clean gray background
- Better user experience
- Comprehensive cost tracking features

## Current Interface
The current application is located in the `frontend/` directory and uses:
- Next.js 14
- React Query for data fetching
- Tailwind CSS for styling
- AWS Amplify for authentication
- Modern component-based architecture

## Notes
These files are kept for reference only and are no longer used in production.
